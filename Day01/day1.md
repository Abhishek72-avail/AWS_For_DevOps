## üíª Cloud Computing Service Models

| Service Model | Description | Examples |
|---------------|-------------|----------|
| **IaaS** (Infrastructure as a Service) | Provides virtualized computing resources like virtual machines, storage, and networking. | AWS EC2, Azure VMs |
| **PaaS** (Platform as a Service) | Provides a platform to develop, run, and manage applications without managing the underlying infrastructure. | AWS Elastic Beanstalk, Heroku |
| **SaaS** (Software as a Service) | Delivers software applications over the internet. | AWS WorkSpaces, Microsoft 365 |

## üìà AWS Account Setup Steps

1. **Visit the AWS website**  
   Go to [https://aws.amazon.com/](https://aws.amazon.com/) and click on **Create an AWS Account**.

2. **Enter Account Information**  
   - Email address
   - Password
   - AWS account name

3. **Provide Contact Information**  
   - Full name
   - Phone number
   - Address

4. **Add Payment Information**  
   - Credit/debit card details
   - AWS might charge a small, refundable amount for verification.

5. **Identity Verification**  
   - AWS may call or text you to confirm your identity.

6. **Select a Support Plan**  
   - Free Basic Plan or a paid plan if needed.

7. **Complete Account Setup**  
   - Sign in to the AWS Management Console using your email and password.

8. **Start Using AWS**  
   - Launch EC2 instances, create S3 buckets, or explore over 200 AWS services!
